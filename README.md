# Industrial-RAD

Industrial-RAD is a research repository for retrieval-augmented diffusion methods in industrial scenarios. It keeps the master project documents, datasets, experiments, metrics, detector code, and supporting scripts in one versioned workspace.

## Repository Layout

```text
Industrial-RAD/
├── docs/
│   ├── MPD/       # Master Project Document sections
│   └── Papers/    # Paper notes and references
├── datasets/      # Dataset manifests and preprocessing notes
├── retrieval/     # Retrieval modules and experiments
├── diffusion/     # Diffusion model components
├── experiments/   # Experiment configs, logs, and reports
├── metrics/       # Evaluation metrics
├── detector/      # Detection models and utilities
└── scripts/       # Reusable project scripts
```

## Getting Started

1. Read `docs/MPD/00_Project_Brief.md` for the current project scope.
2. Track decisions in `docs/MPD/Decision_Log.md`.
3. Keep implementation notes close to the corresponding module directory.
